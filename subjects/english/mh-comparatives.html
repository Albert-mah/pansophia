形容词比较级（taller / bigger / more …）

比较级用来比较两个人或物，常和 than（比）一起出现。
句型：A + be/动词 + 比较级 + than + B。例：Tom is taller than Jim.

🔑 4 条变化规则
情况变法例子
一般情况+ ertall → taller，old → older
以 e 结尾+ rnice → nicer，large → larger
辅音字母 + y去 y，+ ierhappy → happier，heavy → heavier
重读闭音节（辅元辅）双写末尾辅音，+ erbig → bigger，thin → thinner，hot → hotter

多音节 / 较长的词：前面加 more（不加 er）。
  difficult → more difficult，interesting → more interesting，beautiful → more beautiful。

不规则比较级（背下来）

原级比较级
good / well（好的）better
bad（坏的）worse
many / much（多）more
little（少）less
far（远）farther / further
⚠️ 易错点

1. more 和 er 不能同时用：more taller ✗ → taller ✓。
2. 双写别漏：big → bigger，不是 biger。
3. 句中只要比较两者，than 别丢。
4. 想强调"……得多"，用 much / a lot / even + 比较级（much taller），不能用 very。
