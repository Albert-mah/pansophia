26个字母与自然拼读入门

每个英文字母其实有两个身份:一个是它的"名字",一个是它在单词里"干活时发的音"。 先把这两件事分清楚,后面读单词就会顺很多。这一课不背规则,就把字母的名字、字母的发音、还有怎么把音连起来读, 一样一样玩明白。

🔑 一句话先记住

字母名 = 念字母表时它叫什么(像点名);
字母音 = 它躲进单词里发出来的那个声音(像干活)。

读单词的时候,我们用的是字母音,不是字母名。

字母名 vs 字母发音

举个最清楚的例子:字母 B。

📛 它的名字读作 /biː/(像中文"必"),这是你背字母表时念的。

🔧 但它钻进单词 bag(包)里,只发一个短短的 /b/ 音(嘴唇一弹,"不"那个开头), 不会把"必"整个念出来。

所以 bag 不是"必-诶-鸡",而是把三个音 /b/ + /æ/ + /g/ 连起来,读成"拜格"那样的一个词。

再看几个:字母 C 名字读 /siː/,在 cat 里却发 /k/;字母 H 名字读 /eɪtʃ/, 在 hat 里只是轻轻呼一口气 /h/。名字是名字,干活时发的音是另一回事——这就是自然拼读的起点。

💡 怎么区分

背字母表(A B C D…)用的是名字;一个一个去拼读单词时,要把名字换成它的音。 下面的"字母卡"可以一张一张点开看,每个字母的名字和它常发的音都列出来了。

5 个元音字母的短音

英语里 a e i o u 这 5 个字母叫元音字母,是单词的"心脏"——几乎每个单词都得有一个。 它们最常发的是"短音",又快又干脆。先把这 5 个短音读熟,一大半简单单词就能拼出来了。

字母短音像中文哪个音例词(读两遍)
a/æ/咧开嘴的"诶"cat 猫 · bag 包
e/e/短促的"ㄝ / 诶"bed 床 · pen 笔
i/ɪ/短促的"一"sit 坐 · pig 猪
o/ɒ/短促的"哦"dog 狗 · box 盒子

试着把每行的例词慢慢读:c-a-t,先 /k/,再 /æ/,再 /t/,然后越读越快连成 cat。其它几个也照这样拆开再合上。

几个常见的字母组合

有些时候,两个字母会"手拉手"一起发一个新音,这时候别再一个一个拆了,要把它们当成一个整体看。 最常见的有这几组:

sh → /ʃ/,像让人安静时的"嘘"。例:fish 鱼 · ship 船

ch → /tʃ/,像打喷嚏前的"七"。例:chair 椅子 · lunch 午饭

th → 舌尖轻轻咬在上下牙中间送气,有两种(清/浊)。例:think 想 · this 这个

-ck → /k/,出现在词尾,和单个 k 一个音。例:duck 鸭子 · black 黑色

看到 sh / ch / th / ck,提醒自己:这是一组,发一个音,别拆成两半念。

🧩 拼读小诀窍

遇到不认识的词,别急着查、也别去猜整个词长啥样。从左到右,一个音一个音地把字母音读出来,然后越读越快,让它们"连起来", 词的读法常常就自己冒出来了。

比如 map:/m/…/æ/…/p/ → 连起来"麦普" → map; shop:/ʃ/…/ɒ/…/p/ → "夏普" → shop。先慢后快,这就是拼读。

下面给中文提示和例词,你自己把单词读出来,先一个音一个音拆,再连起来。读完心里有数就行:

  1. "猫":先 /k/,再 /æ/,再 /t/ → 连起来读 cat。
  2. "床":/b/ + /e/ + /d/ → bed。
  3. "猪":/p/ + /ɪ/ + /g/ → pig。
  4. "狗":/d/ + /ɒ/ + /g/ → dog。
  5. "杯子":/k/ + /ʌ/ + /p/ → cup。
  6. 挑战:"鱼"——记得 sh 是一组,/f/ + /ɪ/ + /ʃ/ → fish。

读对的标志:每个词都听得出 3 个(或几个)分开的音,最后能流畅连成一个词,而不是去念字母的名字。

🔑 复习一遍

字母有名字(背字母表用)和发音(读单词用),两者不一样。 5 个元音字母 a /æ/、e /e/、i /ɪ/、o /ɒ/、u /ʌ/ 的短音要熟; sh /ʃ/、ch /tʃ/、th、ck /k/ 这些组合发一个音、别拆开。 拼读就一句话:从左到右把字母音读出来,先慢后快连起来。
