一般过去时（动词过去式）

一般过去时表示过去发生的动作或状态。判断要不要用过去时，最简单的办法是找信号词。

🔑 一句话抓住它

句子里出现 yesterday、last…、… ago、just now、this morning、in 2015 这类"过去时间"，动词就要变过去式。

1. 动词怎么变过去式

① 规则动词（直接记 4 条加 ed 规则）

情况变法例子
一般情况+ edplay → played，want → wanted
以 e 结尾+ dlike → liked，live → lived
辅音字母 + y去 y，+ iedstudy → studied，carry → carried
重读闭音节（辅元辅）双写末尾辅音，+ edstop → stopped，plan → planned

② 不规则动词（必须背！小升初高频）

原形 → 过去式原形 → 过去式原形 → 过去式
go → wentcome → camesee → saw
do → didhave → hadget → got
make → madetake → tookeat → ate
run → ranswim → swambuy → bought
write → wroteread → read（音变）say → said
tell → toldgive → gavethink → thought

2. 否定句和疑问句

✅ 一变就对

• 否定句：主语 + didn't + 动词原形。
  I didn't went → I didn't go to school. （didn't 后面一定用原形！）

• 一般疑问句：Did + 主语 + 动词原形…？ — 回答 Yes, I did. / No, I didn't.

• be 动词的过去时：am/is → was，are → were；否定直接加 not（wasn't / weren't），疑问把 was/were 提前。

⚠️ 最容易丢分的 3 个坑

1. 已经有 did / didn't，后面动词不能再变过去式（用原形）。
2. 规则动词别忘了 ed；不规则动词别"想当然加 ed"（go→goed ✗）。
3. 看到 yesterday 等信号词却用了一般现在时——先圈出信号词再写。

✏️ 马上检测一下

→ 不规则动词过去式默写
→ 一般过去时句子练习
