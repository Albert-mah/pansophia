this / that / these / those：指示代词

英语里,当我们要"指着一样东西说话"的时候,就会用到这四个词: this / that / these / those。它们叫指示代词—— 就是"用来指东西的词"。选哪一个,只看两件事:东西离我远还是近, 东西是一个还是很多个。把这两件事搞清楚,就全会了。

🔑 一句话记住

近的用 this / these,远的用 that / those;
一个用 this / that,多个用 these / those。

小窍门:带 -s 结尾的 these / those 都是复数(多个), 就像名词复数也常常带 -s 一样,顺口又好记。

2×2 一张表全看懂

把"远/近"放成两行,"单数/复数"放成两列,四个词正好填满四个格子:

单数(一个) 复数(多个)
近(在我手边) this 这个 these 这些
远(在那边) that 那个 those 那些

比如:手里捏着一支笔,说 this pen(这支笔);讲台上摆着一排笔, 离你远,又是好几支,就说 those pens(那些笔)。

be 动词要"对得上":is 还是 are?

指示代词后面的 be 动词,必须跟着"单数还是复数"一起变:

  • 单数配 is:this / that 后面用 is。
  • 复数配 are:these / those 后面用 are。

This is a pen. —— 这是一支笔。

These are pens. —— 这些是笔。

That is a dog. —— 那是一只狗。

Those are dogs. —— 那些是狗。

注意:从单数变复数,不光把 this 换成 these, is 也要换成 are,名词也要加 -s(pen → pens, dog → dogs)。三处一起变,才算对。

提问和回答

想问"这/那是什么",句型也分单复数。回答的时候,用 it(单数)或 they(复数)来代替前面问的东西:

What's this? —— 这是什么?
— It's a ruler. —— 它是一把尺子。

What's that? —— 那是什么?
— It's an apple. —— 它是一个苹果。

What are these? —— 这些是什么?
— They're books. —— 它们是书。

What are those? —— 那些是什么?
— They're birds. —— 它们是鸟。

小提示:What's = What is(单数问), It's = It is,They're = They are(复数答)。问什么数,就用什么数回答。

这些孩子最常错的两点

❌ 错: This are my friends.
✅ 对: These are my friends. —— 这些是我的朋友。
原因:friends 是复数(好几个朋友), 不能用单数的 this,要用 these。

❌ 错: These is my pens.
✅ 对: These are my pens. —— 这些是我的笔。
原因:these 是复数,后面必须配 are,不能用 is。

❌ 错: That are nice flowers.
✅ 对: Those are nice flowers. —— 那些是好看的花。
原因:flowers 是复数又在远处,要用 those + are。

记住一条检查法:说完一句,先看名词是不是复数(有没有 -s), 再回头检查代词和 is/are 有没有"跟上"。三处对齐,基本不会错。

✏️ 试一试

填上 this / that / these / those,或选对 is / are;最后两题把中文翻成英文:

  1. ______ is my schoolbag. (这个,在我手上)
  2. ______ are your shoes. (那些,在门口那边)
  3. These ______ (is / are) red apples.
  4. 请翻译:这些是我的同学。 ______
  5. 请翻译:那是一只猫。 ______

答案:1. This　2. Those　3. are 4. These are my classmates.　5. That is a cat.(也可写 That's a cat.)

🔑 复习一遍

近单 this,近多 these;远单 that,远多 those。 单数配 is,复数配 are。问 What's this/that 用 It's 答, 问 What are these/those 用 They're 答。每说一句,都顺手检查一下名词、代词、be 动词三处的"单复数"对没对上。
