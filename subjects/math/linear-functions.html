一次函数与正比例

初中数学里，函数是描述"量与量之间关系"的工具。一次函数是其中最简单、最重要的一种——它的图象就是一条直线，所以也叫线性函数。只要搞懂 k 和 b 这两个数，这条直线的一切你都能掌控！

🔑 核心一句话

一次函数 $y = kx + b$（$k \neq 0$）就是一条直线：

  • k（斜率）——决定这条线有多陡、往哪个方向倾斜；
  • b（截距）——这条线在 y 轴上的"起点"，即 $x = 0$ 时 $y = b$。

记住：k 和 b 确定了，直线就完全确定了，一条都不多，一条都不少。

1. k 是什么——斜率的几何意义

k 告诉我们直线的倾斜程度和方向。图里绿色虚线小三角形就是 k 的形象：

  • 向右走 1 格，y 变化 k 格。k 绝对值越大 → 线越陡；k 越接近 0 → 线越平。
  • $k > 0$：直线从左下到右上，函数单调递增（x 越大，y 越大）。
  • $k < 0$：直线从左上到右下，函数单调递减（x 越大，y 越小）。
  • $k = 0$：直线水平，不是一次函数——定义里特别要求 $k \neq 0$！

还有一个很有用的结论：两条直线平行，则它们的斜率相等（且截距不同）。比如 $y = 2x + 1$ 和 $y = 2x - 3$ 是两条平行线。

2. b 是什么——y 轴截距的几何意义

把 $x = 0$ 代入 $y = kx + b$，得到 $y = b$。所以 b 就是直线与 y 轴交点的纵坐标，这个点是 $(0,\ b)$，叫做 y 轴截距（实验里红色圆点）。

  • $b > 0$：交点在原点上方。
  • $b < 0$：交点在原点下方。
  • $b = 0$：直线经过原点，这就变成了正比例函数。

3. 图象与坐标轴的两个交点

画直线时，我们只需要两个点就够了。以下两个交点最容易算：

交点求法坐标
与 y 轴的交点令 $x = 0$，得 $y = b$$(0,\ b)$
与 x 轴的交点 令 $y = 0$，解 $kx + b = 0$ $\left(-\dfrac{b}{k},\ 0\right)$

找到这两个点，用直尺一连，直线就画好了。

✏️ 例题：已知两点，求 k 和 b

题：已知直线 $y = kx + b$ 经过点 $A(1,\ 3)$ 和 $B(-1,\ -1)$，求 $k$、$b$。

解：把 A、B 两点的坐标分别代入 $y = kx + b$，列出方程组：

$$\begin{cases} k \times 1 + b = 3 \\ k \times (-1) + b = -1 \end{cases} \implies \begin{cases} k + b = 3 \quad \cdots ①\\ -k + b = -1 \quad \cdots ②\end{cases}$$

① $-$ ②，消去 b：$2k = 4$，解得 $k = 2$。

把 $k = 2$ 代回①：$2 + b = 3$，解得 $b = 1$。

结果：$k = 2$，$b = 1$，函数解析式为 $y = 2x + 1$。

验算：把 $B(-1, -1)$ 代入：$2 \times (-1) + 1 = -1$ ✓　放心，验算是好习惯！

💡 正比例函数是一次函数的"特例"

正比例函数 $y = kx$（$k \neq 0$）其实就是 $b = 0$ 的一次函数，特别之处是：图象一定经过原点 $(0, 0)$。

类比记忆：正比例函数 ⊂ 一次函数，就像正方形 ⊂ 长方形——前者是后者的特殊情况。

判断口诀：有 b = 0 且过原点，正比例；b ≠ 0 不过原点，一次函数。
